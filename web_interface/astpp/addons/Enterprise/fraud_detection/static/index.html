Key Features

User interface

  • Block user access on exceeded attempt to login with wrong credentials
  • Configure limit per account per day for Daily credit/ Recharge
  • Configure limit for global recharge per day
  • Configuration to allow unlimited recharge and credit

Actions taken

  • Send an alert if the value goes above a set value
  • Block operation if the value goes above set block value

Call

  • Configure per day limit for
      Maximum allowed number of calls per destination
      Maximum allowed overall call cost per destination
      Maximum allowed number of calls per account

Other Features

      Override set limits for selected customer accounts
      IP based Whitelisting
      IP based blacklisting
