Key Features

Multi-tenant support

  • Each customer will be considered as tenant
  • Allow to configure domain name

Conference

  • Able to create conferences
  • Secure conference using pin usage
  • Allocate DID number to use conference
  • Customer can purchase DID from customer portal and map it to conference to make it publically available

Ring Group

  • Able to create number of ring groups
  • Simultaneous and Sequential ring strategies supported
  • Digit confirmation for external agents
  • DID number allocation (Same as conference)

Queue

  • Able to create number of queues
  • Configurable ringback tones for agent and customers
  • Separate number configuration for agents to login and logout
  • DID number allocation (Same as conference)

IVR

  • Able to create number of IVRs
  • Configurable caller name & number prefix for easy IVR call identification
  • Configurable invalid retries
  • Option to allow direct dial
  • Configurable ring back tone
  • Configurable IVR menus

Recording

  • Create recording using uploading file or by dialing number
  • Allow to play and download recording file

Time Conditions

  • Able to create number of time conditions
  • Make DID destination routing based on time conditions
  • Can create multiple internal conditions for routing

Phone Book

  • System-wide speed dial numbers that can be dialed from any phone

SIP Extensions

  • Configurable Caller name and number
  • Enable or disable voicemails
  • Option to enable or disable call recording
  • Follow me (Including Call Forwarding)
  • DND
  • Pickup Group configuration
