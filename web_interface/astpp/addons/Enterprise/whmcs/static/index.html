Key Features

  • Create customer from WHMCS by purchasing product with various configurations
    1. Define Account Type (Prepaid, Postpaid)
    2. Allocate Initial Balance
    3. Allocate Credit Limit for Postpaid customers
    4. Default Tariff Plan
    5. Enable/Disable permissions for IP Management, Call Recording and Local Calls
    6. Invoice Configurations
  • Easy Customer management (Suspend, Unsuspend and Terminate) from WHMCS
  • Recharge Customer account using WHMCS Product
  • Invoice synchronization between WHMCS and ASTPP
  • Extend payment options by using any WHMCS supported payment gateway
